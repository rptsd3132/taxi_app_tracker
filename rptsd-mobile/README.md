# RPTSD Mobile App

Android native app using Kotlin + Jetpack Compose.

## Setup
Open this folder in Android Studio (on Windows) and build from there.

## Build Locally
```bash
./gradlew assembleDebug
```

## Run on Emulator
```bash
./gradlew installDebug
adb shell am start -n com.simplenotes.helper/.MainActivity
```
